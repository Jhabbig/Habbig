# gateway/static/img/ — Image assets

Static images referenced from the gateway HTML pages and CSS. Anything
the browser actually loads as an `<img src>` or `background-image:
url(...)` lives here.

## Files in this directory

| File | Purpose |
|---|---|
| `logo.png` | Narve brand logo. Used in the top nav, favicon link, and OG share previews. |
| `tobias.jpg` | Founder headshot used on the landing page / impressum / about section. |
