Polymarket
{{ raw_poly_status }}
Trade prediction markets on Polymarket using your existing Ethereum wallet. We only need your wallet's private key.
Where do I find my private key? (MetaMask, 30 seconds)
  1. Open the MetaMask browser extension and click your account icon (top right circle).
  2. Click the three-dot menu next to your account name, then choose Account details.
  3. Click the Show private key button.
  4. Enter your MetaMask password to unlock it.
  5. Copy the long string of letters and numbers, then paste it into the field above.
Important — use a dedicated trading wallet. Don't paste your main wallet's private key. Create a fresh wallet, fund it only with the USDC you intend to trade, and use that key here. If anything ever goes wrong, only the funds in that one wallet are at risk.

Don't have a Polymarket wallet yet? Sign up at polymarket.com — they'll walk you through creating an embedded wallet you can export the key from.

Advanced — I already have CLOB API keys

Most users should leave this section alone. We'll auto-derive a CLOB API key from your private key on your first trade. Only fill these in if you've already minted CLOB credentials elsewhere and want to reuse them.

Trade event contracts on Kalshi. Pick whichever option below is easier — both work the same.
A API Keyrecommended
How do I get a Kalshi API key?
  1. Sign in to your account at kalshi.com.
  2. Click your profile icon in the top right, then API Keys (or visit Account → Profile).
  3. Click Generate New Key and give it a name like "narve.ai".
  4. Copy the key Kalshi shows you and paste it into the field above.

Don't have a Kalshi account? Sign up here — it takes a few minutes including ID verification.

B Email + Password

Trade US stocks, ETFs, and crypto through your Alpaca brokerage account. Paper trading is enabled by default — switch to live when you're ready.
1 API Key + Secret
How do I get Alpaca API keys?
  1. Sign up at alpaca.markets (free).
  2. In the dashboard, go to Paper Trading (or Live if funded).
  3. Click View API Keys, then Generate New Key.
  4. Copy the API Key ID and Secret Key and paste them above.

We recommend starting with Paper Trading keys to test risk-free with virtual funds. You can switch to live keys later in your Alpaca dashboard.
