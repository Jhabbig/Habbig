Changelog

What's shipped on narve.ai.

21 April 2026

SEO content pages + 90-day strategy

Added dedicated pages for /about, /how-it-works, /methodology, /faq, /team, /press, and /changelog, each with its own schema.org JSON-LD, canonical URL, and Open Graph metadata. Full 90-day SEO plan committed at SEO_STRATEGY.md.

21 April 2026

Subproduct marketing pages + admin MRR

Branded marketing pages for the six narve.ai subdomains — sports, weather, world, crypto, midterm, traders — plus an admin MRR view broken down per subproduct.

21 April 2026

Security remediation pass

First pass on the non-bypassable access audit: tightened host validation, hardened impersonation privilege checks to fail-closed when admin_level is absent, and expanded the impersonation block list to include 2FA and API-key paths on all methods.

20 April 2026

SEO foundation

robots.txt and sitemap.xml now served from the gateway with per-path changefreq and priority. Added a dedicated /narve brand-query landing page. Full Organization + WebSite + SoftwareApplication JSON-LD blocks on the homepage.

19 April 2026

Admin: impersonation, feature flags, email template editor

Three admin tools shipped together:

  • User impersonation with reason-required, full audit trail, and a destructive-action block list.
  • Feature flags with per-user, per-tier, and percentage-rollout controls.
  • Admin-editable email templates with live preview; broken overrides fall back to the file template so emails never drop.
18 April 2026

In-app notification bell

Bell icon in the top nav with unread badge. Dropdown shows the last ten, with "Mark all read" and "View all." Server-Sent Events for real-time updates. Per-type preferences page at /settings/notifications.

17 April 2026

Embed widgets

Three embeddable widget types — source credibility, market probability, and best bets — with per-widget domain scoping and Referer validation. Creation flow at /settings/embeds, paid users only, max ten widgets per account.

16 April 2026

Affiliate program

Admin-provisioned affiliate accounts with per-affiliate commission rates. 90-day attribution cookie. Payouts manually approved via the admin panel; no Stripe Connect. First affiliates onboarded by end of month.

15 April 2026

Private referrals + leaderboard

Each paid user gets a referral link. Rewards: 1 paid referral = +1 month free, 5 = +1 tier upgrade, 10 = 3 months Pro. Accuracy leaderboard at /leaderboard is opt-in for subscribers only. No public "refer a friend" CTAs.

14 April 2026

User predictions

Users can now submit their own predictions on any active market (YES/NO + probability). 24h edit window, direction locked. /predictions page shows personal history + Brier + timing + per-category stats. Opt-in public profile at /predictions/public/{user_id}.

Older entries live in the repo migration history. For technical release notes, see methodology or the public GitHub mirror.
