# app/desktop/assets/ — Desktop app icons

Icon files used by the macOS desktop wrapper. Generated by
`scripts/generate_icons.py` (run as part of `scripts/build_dmg.sh`); don't
hand-edit them.

## Files in this directory

| File | Purpose |
|---|---|
| `icon.icns` | macOS app icon — bundled into the `.app`'s `Contents/Resources/`. |
| `icon.png` | 512×512 PNG source icon — used for in-app references that need a raster image. |
| `menubar_icon.png` | Small monochrome PNG used by `menu_bar.py` as the rumps `template=True` icon (system auto-tints it for light/dark mode). |
