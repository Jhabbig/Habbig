# Third-party data files

The following files are vendored from the **worldmonitor/worldmonitor** repo
(<https://github.com/worldmonitor/worldmonitor>), licensed under Apache-2.0:

| File | Source path | Purpose |
|------|-------------|---------|
| `country-bboxes.json` | `scripts/shared/country-bboxes.json` | Country bounding boxes for map zoom-to-country |
| `iso2-to-iso3.json`   | `scripts/shared/iso2-to-iso3.json`   | Country code mapping (alpha-2 → alpha-3) |
| `iso3-to-iso2.json`   | `scripts/shared/iso3-to-iso2.json`   | Country code mapping (alpha-3 → alpha-2) |

The remaining files in this directory (`maplibre-gl.*`, `chart.umd.min.js`,
`countries.geojson`, `favicon.png`) are vendored from upstream open-source
projects. See the LICENSE files in those projects for terms.
