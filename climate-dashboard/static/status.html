"Down" means the upstream URL returned no data — usually a stale URL after a host restructure. "Error" means the fetch raised an exception. Both surface in this list so the operator can swap URLs without poking at individual endpoints. The dashboard's own cards render an explicit "data unavailable" placeholder for any source that's down here.
